# 📊 Pipeline de Análise de Dados - E-commerce

Este projeto apresenta uma pipeline de análise de dados desenvolvida em Python no Google Colab, simulando um cenário de e-commerce com grande volume de informações.

---

## 🎯 Resumo do Projeto

Neste projeto, foi desenvolvida uma pipeline de análise de dados utilizando Python para processar informações de um e-commerce fictício.

A principal descoberta foi que bibliotecas como **Pandas** e **NumPy** permitem processar grandes volumes de dados de forma muito mais eficiente do que métodos tradicionais baseados em loops.

Além disso, foram extraídos insights como:
- Valor médio de compras por hora
- Produtos mais visualizados

---

## 🧩 Contexto do Problema

A empresa simulada é uma plataforma de e-commerce que gera grande volume de dados diariamente, como visualizações, compras e interações de usuários.

O setor de Marketing enfrentava dificuldades para analisar esses dados devido ao uso de processos manuais e lentos.

O desafio foi atuar como Engenheiro de Dados e desenvolver uma solução eficiente para análise de alto volume de dados.

---

## ⚙️ Análise e Abordagem

Foram utilizados conceitos de:

- Programação vetorial
- DataFrames (Pandas)
- Arrays (NumPy)

Essas técnicas substituem loops tradicionais, tornando o processamento mais rápido e escalável.

Principais operações realizadas:
- Filtragem de dados (compras e visualizações)
- Agrupamento por hora (groupby)
- Cálculo de métricas e agregações

---

## 📈 Resultados

A pipeline gerou duas análises principais:

### 🔹 Valor médio de compras por hora
Permite identificar os horários com maior valor de compra, ajudando na definição de estratégias de marketing.

### 🔹 Produtos mais visualizados
Mostra quais produtos recebem mais atenção dos usuários, auxiliando em campanhas, destaques e promoções.

Esses insights ajudam a transformar dados brutos em decisões estratégicas.

---

## 🧠 Conclusão

O projeto reforçou a importância da Engenharia de Dados em cenários de Big Data.

Foi possível entender que o uso de ferramentas como Pandas e NumPy não é apenas organização, mas sim desempenho e escalabilidade.

Também ficou claro como a análise de dados contribui diretamente para decisões de negócio.
